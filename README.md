# PlenkaGeminiBot

Простой Telegram-бот на Python с Gemini API. Поддерживает:
- выбор текстовой модели через `/model`:
  - `Gemini 2.5 Flash`
  - `Gemini 3 Flash`
  - `Qwen Next 80B`
  - `Kimi K2 Thinking`
  - `MiniMax M2.1`
- генерацию картинок через `/img`
- доступ только для владельца и добавленных пользователей (список хранится в `users.db`)

## Быстрый старт

1. Создайте виртуальное окружение и установите зависимости.
2. Скопируйте `.env.example` в `.env` и заполните значения.
3. Запустите бота.

Команды:
- `/start` — приветствие
- `/whoami` — узнать свой `user_id`
- `/allow <user_id>` — добавить пользователя (только владелец). Также можно ответить на сообщение пользователя.
- `/deny <user_id>` — удалить пользователя (только владелец). Также можно ответить на сообщение пользователя.
- `/list` — список пользователей с доступом (только владелец)
- `/model` — открыть меню выбора текстовой модели
- `/img <описание>` — сгенерировать картинку
- `/memory on|off|status` — включить/выключить память диалога для текущего чата
- `/clear` — очистить историю (память) текущего чата

## Настройки

Все настройки через переменные окружения, см. `.env.example`.
Можно указать `OWNER_USERNAME`, чтобы владелец определялся по @username (в дополнение к `OWNER_USER_ID`).
Для памяти используется SQLite (история хранится в `users.db`).

## Запуск

```powershell
python -m venv .venv
.venv\\Scripts\\Activate.ps1
pip install -r requirements.txt
copy .env.example .env
python main.py
```
